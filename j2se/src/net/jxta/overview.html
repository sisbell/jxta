These packages provide the standard API for the JXTA Protocols using Java Standard Edition 5.0. The implementation is divided into two parts: the JSE Standard JXTA API (these packages) and the JSE JXTA Reference Implementation

Important Note: Applications and services are very strongly encouraged to not import classes or interfaces from the implementation packages. Specificially do not import from net.jxta.impl.*. The content of these packages may change significantly (including removal) from release to release with no advance notice. @see JXTA 2.0 Protocols Specification
