The JXSE Project provides a complete reference implementation of the JXTA protocols built using Java Standard Edition (JSE) 5.0. Binary images of the current release and all previous releases can be downloaded at JXTA Downloads. This image contains the protocol implementation and a sample Shell application (shell.jxta.org) that permits you to interactively experiment with JXTA.

The JXTA Protocols are specified by the JXTA Protocols Specification Project (jxta-spec.dev.java.net). The JXTA Protocols Specification Project is charged with producing a formal and complete specification for the JXTA protocols.
